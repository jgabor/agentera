# D47 App-Home Vocabulary Inventory

This inventory classifies repository occurrences of bundle-era vocabulary and legacy
artifact labels for Decision 47 follow-up work. It is intentionally scoped to
classification only; implementation belongs to a later task.

## Search Coverage

- Terms searched: `bundle`, `bundled`, `bundle-root`, `bundle_root`, `bundle root`, `install root`, `DECISIONS.md`, `DOCS.md`, `PLAN.md`, `PROGRESS.md`.
- Surfaces covered: repository Markdown/YAML/JSON/Python/shell files under `/home/jgabor/git/agentera`, including root docs, `.agentera/*.yaml`, `skills/agentera/**`, `scripts/**`, `hooks/**`, `references/**`, `tests/**`, plugin/package manifests, runtime adapter metadata, and package/runtime contract schemas. Classification is grouped by path/pattern and contract role; this is not a line-by-line exhaustive rewrite plan.
- Excluded from planned replacement by default: historical decisions, changelog entries, explicit legacy-migration examples, compatibility test fixtures, serialized/public identifiers, and negative validators that intentionally detect stale wording.
- Excluded from classification: vendored/cache dependency trees such as `.venv`, `.pytest_cache`, and `.opencode/node_modules`.
