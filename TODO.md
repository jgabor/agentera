- [ ] [fix:3.0.0] Shim `isRunnableFile` has no shebang/runtime validation or fallback. `resolve.mjs:50-57` only checks `stat.isFile()`. Cannot distinguish a Python PEP 723 script from a Node script. (Branch: `feat/v3` — implement on `feat/v3`.)
